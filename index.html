Image Steganalysis: Detect Hidden Information with ML
Bitcamp Hackathon (Apr 2025)

Technologies: Python, scikit-learn, XGBoost, HOG/entropy features

Built a comprehensive machine learning system to detect hidden messages in images using steganography detection techniques. The project involved creating a reproducible Python pipeline that preprocessed image data, extracted HOG and entropy-based features, and benchmarked multiple algorithms including Logistic Regression, Random Forest, and XGBoost.

Key achievements include achieving high AUC and PR-AUC scores across different models, implementing feature importance analysis for model explainability, and delivering a complete solution within the 36-hour hackathon timeframe. The pipeline was designed with reproducibility in mind, featuring clear documentation and modular code structure.

This project demonstrated proficiency in computer vision techniques, machine learning model comparison, and rapid prototyping under time constraints.
